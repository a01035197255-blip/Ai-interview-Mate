## 📌 프로젝트 소개

InterviewMate는 사용자의 지원서를 기반으로 AI 모의면접을 진행하고,
답변에 대한 피드백을 제공하여 실제 면접을 대비할 수 있도록 돕는 서비스입니다.

---

## ✨ 주요 기능

### 📝 지원서 관리

* 지원서 등록
* 수정 및 삭제
* 지원서 조회

### 🤖 AI 모의면접

* 직무 기반 질문 생성
* 면접 진행
* 답변 저장

### 📊 AI 피드백

* 답변 분석
* 개선 포인트 제안
* 종합 평가 제공

### 📂 면접 이력 관리

* 과거 면접 조회
* 답변 기록 확인
