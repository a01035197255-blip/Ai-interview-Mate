# 🚀 InterviewMate

### AI 기반 모의면접 및 지원서 평가 플랫폼

AI 면접 · 지원서 관리 · 맞춤형 피드백

---

## 🌐 배포 주소
https://interviewmate.n-e.kr/

---

## 📌 프로젝트 소개

InterviewMate는  
**지원서 관리 기능**과 **직무 설정 기반 AI 모의면접 기능**을 제공하는  
취업 준비 지원 플랫폼입니다.

두 기능은 독립적으로 동작하며, 사용자는 각각의 목적에 맞게 활용할 수 있습니다.

---

## 🔥 SYSTEM FLOW

### 📝 지원서 관리 시스템 (CRUD)

지원서 생성 → 조회 → 수정 → 삭제  
↓  
지원서 데이터 저장 및 관리  

---

### 🎤 AI 모의면접 시스템 (직무 기반)

직무 설정  
↓  
AI 질문 10개 생성  
↓  
면접 진행 (질문당 5분 제한)  
↓  
사용자 답변 저장  
↓  
AI 답변 평가  
(논리성 / 키워드 / 직무 적합도)  
↓  
질문별 피드백 생성  
↓  
종합 면접 리포트  

---

## ✨ 주요 기능

### 🔐 인증 및 사용자 관리
- 이메일 인증 (6자리 코드)
- 회원가입 / 로그인
- JWT 기반 인증 / 인가
- Spring Security 적용

---

### 🌐 소셜 로그인
- Google OAuth
- Naver OAuth
- 자동 회원가입 및 로그인

---

### 📝 지원서 관리 (독립 기능)
- Create / Read / Update / Delete
- 지원서 개별 관리

---

### 🤖 AI 모의면접 (직무 기반)
- 직무 선택 기반 질문 생성
- AI 질문 10개 생성
- 면접 진행 및 답변 저장

---

### 📊 AI 피드백
- 답변 분석
- 개선 포인트 제공
- 종합 평가

---

### 📂 면접 이력 관리
- 과거 면접 기록 조회
- 답변 기록 확인
