# 🚀 InterviewMate

### AI 기반 모의면접 및 지원서 평가 플랫폼

AI 면접 · 지원서 분석 · 맞춤형 피드백

🌐 배포 주소  
https://interviewmate.n-e.kr/

---

## 📌 프로젝트 소개

InterviewMate는 사용자의 직무 설정을 기반으로  
AI 모의면접을 진행하고 답변에 대한 피드백을 제공하여  
실제 면접을 대비할 수 있도록 돕는 서비스입니다.

---

## 🔥 SYSTEM FLOW

📝 지원서 관리 시스템 (CRUD)
→ Create / Read / Update / Delete  
↓  
💾 지원서 데이터 저장  
↓  
🧠 AI 지원서 기반 피드백  
→ 이력서 분석  
→ 자기소개서 분석  
→ 직무 적합도 평가  
↓  
🎤 AI 모의면접 시스템  
→ 직무 설정  
→ AI 질문 10개 생성  
→ 면접 진행  
↓  
⏱️ 질문당 제한시간 5분  
→ 사용자 답변 저장  
↓  
📊 AI 답변 평가  
→ 논리성 분석  
→ 키워드 분석  
→ 직무 적합도 분석  
↓  
🧾 질문별 피드백 생성  
↓  
📑 종합 면접 리포트

## ✨ 주요 기능

### 🔐 인증 및 사용자 관리
- 이메일 인증 (6자리 코드 기반 인증)
- 회원가입 및 로그인 기능
- JWT 기반 인증 및 인가 처리
- Spring Security 기반 보안 구조 적용
- 사용자 세션 및 권한 관리

---

### 🌐 소셜 로그인
- Google OAuth 로그인
- Naver OAuth 로그인
- 소셜 로그인 기반 자동 회원가입 및 로그인 처리
- 기존 계정과 소셜 계정 연동 지원

---

### 📝 지원서 관리
- 지원서 등록 (Create)
- 조회 (Read)
- 수정 (Update)
- 삭제 (Delete)

---

### 🤖 AI 모의면접
- 직무 기반 질문 생성
- 면접 진행
- 답변 저장

---

### 📊 AI 피드백
- 답변 분석
- 개선 포인트 제안
- 종합 평가 제공

---

### 📂 면접 이력 관리
- 과거 면접 조회
- 답변 기록 확인
